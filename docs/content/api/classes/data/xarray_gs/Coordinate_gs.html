Attach a coordinate/dimension to a Dataset from a dictionary.

If ‘bounds’ are not defined inside the dict, the coordinate is discrete. Otherwise, bounds are attached also in order to satisfy ???

Parameters

  • name (str) – The name of the coordinate to attach

  • standard_name (str) – CF convention standard name

  • long_name (str) – CF convention long name

  • units (str) – units of the coordinate

  • null_value (int or float) – what are null values represented by

  • centers (array_like) – Has shape (size of dimension, ) defining the center values of each “cell”

  • bounds (array_like, optional) – Has shape (size of dimension, 2) defining the bounds of each “cell” in the coordinate
